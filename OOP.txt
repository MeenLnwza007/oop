OOP Differences
เปรียบเทียบความเหมือนและความแตกต่างในการเขียนโปรแกรมแบบ OOP ด้วยภาษา Java, C++, Python
ความเหมือน
การสร้างวัตถุจากคลาส - สร้างวัตถุเพื่อเป็นตัวแทนของคลาสในการเรียกใช้งาน
สนับสนุนการสืบทอด (Inheritance) - สามารถสืบทอดจากคลาสแม่ไปยังคลาสลูกได้
สนับสนุนการห่อหุ้ม (Encapsulation) - ฟิลด์หรือเมธอดมี access modifiers สำหรับควบคุมการเข้าถึง
การใช้ Method Overriding - สามารถเรียกใช้เมธอดชื่อเดียวกันในต่างคลาส และช่วยให้เกิดพอลิมอร์ฟิซึม
สนับสนุนการพอลิมอร์ฟิซึม (Polymorphism) - เมธอดเดียวกันแต่การทำงานต่างกัน
การส่ง Message - ด้วยการใช้ . เพื่อเรียกใช้เมธอดผ่านวัตถุ หรือใช้ -> เพื่อเรียกใช้เมธอดผ่าน pointer
ความแตกต่าง
 การคืนพื้นที่หน่วยความจำ (De-allocate) 
- Java และ Python: มี garbage collector ช่วยจัดการหน่วยความจำให้อัตโนมัติ โดยโปรแกรมเมอร์ไม่จำเป็นต้องเรียกใช้หรือจัดการด้วยตนเอง
- C++: ไม่มี garbage collector ทำให้ต้องจัดการคืนหน่วยความจำด้วยตนเอง ด้วยการใช้คำสั่ง delete
 การประกาศ Attributes และ Methods 
- Java และ C++:  การสร้างและเรียกใช้ ต้องกำหนดประเภทข้อมูลให้ชัดเจน
- Python: ไม่จำเป็นต้องกำหนดประเภทข้อมูลก่อน
 การใช้ Constructor และ Destructor 
- Java: มี constructor สำหรับสร้างอินสแตนซ์ของคลาส แต่ไม่มี destructor ต้องให้ garbage collector จัดการหน่วยความจำให้เองอัตโนมัติ
- C++: มีทั้ง constructor และ destructor ด้วยการใช้ ~ (tilde) หน้าคลาสนั้น ๆ
- Python: มี constructor ด้วยการใช้ __init__(self) และมี destructor ด้วยการใช้ __del__(self)
